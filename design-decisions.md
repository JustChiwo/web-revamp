# Business & Site Context

- Bennett Tea — an e-commerce site selling curated, specific types of tea (their niche/specialty is the specific tea varieties they sell)

## Original Site Weaknesses

- Typography — the font felt harsh and unwelcoming, not clean or inviting for a tea brand
- Navigation — not user-friendly; menu structure required extra steps/thought for users to find what they needed (e.g. sections tucked into a menu rather than being immediately visible)

## Design Decisions & Fixes
--Why this business:
--We chose Bennett Tea because the original site didn't look visually appealing, and we personally experienced difficulties navigating it, which made it a strong candidate for a redesign.

## UX Improvements

- Font/typography — switched to a softer, more aesthetic font to match the calming, "tea" mood/brand feel
- Navigation bar — redesigned to be persistent/visible (e.g. About, Contact) rather than hidden inside a collapsed menu — so users can click directly to a section (like "About") with no extra effort
- Direct-access navigation — clicking "About" (or similar links) takes users straight to that section with zero ambiguity or extra clicks

## Bootstrap Features Used:

- Navbar — Bootstrap navbar as the base, customized with our own CSS
-Contact page — Bootstrap form fields for the contact form
- Products page — Bootstrap title cards for displaying tea products
- About page — mainly custom CSS, minimal Bootstrap

## Custom CSS:

-Applied throughout for branding — typography (softer font), navbar styling, and the About page layout
